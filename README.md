# Portfolyo Website 👩‍💻🌸🎀

Bu proje, React ve Vite kullanılarak geliştirilmiş bir kişisel portföy uygulamasıdır.  
Tasarım, verilen UI yönergelerine  uyularak uygulanmıştır. Geliştirme ve güncellemeler devam etmektedir.

---

## Proje Özellikleri

- React + Vite tabanlı modern frontend mimarisi
- TailwindCSS ile component bazlı stil yönetimi
- Dark / Light tema desteği (Context API + LocalStorage)
- Türkçe / İngilizce çok dilli yapı
- Statik JSON dosyaları üzerinden içerik yönetimi
- Context API ile global state yönetimi
- Axios kullanılarak API servisi ile iletişim


---

## Kullanılan Teknolojiler

- React
- Vite
- TailwindCSS
- Axios
- Context API
- JavaScript
